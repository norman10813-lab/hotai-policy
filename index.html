首次使用：請先完成以下三個步驟，設定 Google Sheets 連線後即可啟用雲端同步與管理者功能。
1建立 Google Sheets 與 Apps Script
  1. 前往 sheets.google.com 建立新試算表
  2. 點選上方「擴充功能」→「Apps Script」，將 Code.gs 內容完整貼入並儲存
  3. 修改第 4 行 ADMIN_PASSWORD 設定你的管理者密碼
2部署為 Web App
  1. Apps Script 右上角點「部署」→「新增部署作業」
  2. 類型選「網路應用程式」，執行身分選「我」，存取權選「所有人」
  3. 點「部署」，複製產生的 Web App 網址
3貼入設定並初始化資料
Web App 網址與密碼僅存在你自己的瀏覽器中（localStorage），不會傳給任何人。夥伴只需開啟此 HTML 檔，無需知道密碼即可正常使用所有試算功能。
